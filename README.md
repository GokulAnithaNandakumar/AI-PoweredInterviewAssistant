## üåü **Project Architecture**

This project is a full-stack AI-powered interview assistant with:
- **React + Vite + TypeScript** frontend (dashboard & candidate chat)
- **FastAPI (Python)** backend (REST API & WebSocket)
- **Google Gemini + LangGraph** for AI question/answer/summary

## üîå **API Endpoints**

### **Authentication**
- `POST /api/auth/register` ‚Äî Register interviewer
- `POST /api/auth/login` ‚Äî Login interviewer
- `POST /api/auth/create-session` ‚Äî Generate candidate link (with role)

### **Interview (Candidate)**
- `GET /api/interview/{token}/info` ‚Äî Get session info (name, email, phone, role, status, etc)
- `POST /api/interview/{token}/upload-resume` ‚Äî Upload resume (PDF/DOCX)
- `PUT /api/interview/{token}/candidate-info` ‚Äî Update candidate info (missing fields)
- `POST /api/interview/{token}/start-interview` ‚Äî Begin interview
- `POST /api/interview/{token}/submit-answer` ‚Äî Submit answer

### **Dashboard (Interviewer)**
- `GET /api/dashboard/sessions` ‚Äî List all sessions for interviewer
- `GET /api/dashboard/sessions/{id}/details` ‚Äî Get full candidate details (including role)
- `GET /api/dashboard/stats` ‚Äî Dashboard statistics
- `DELETE /api/dashboard/sessions/{id}` ‚Äî Delete a session

### **WebSocket**
- `WS /ws/{session_token}` ‚Äî Real-time chat/interview updates
